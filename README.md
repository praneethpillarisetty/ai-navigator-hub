# AI Navigator Hub

Static SEO-friendly AI tools directory for Cloudflare Pages.

## Deploy on Cloudflare Pages

1. Upload this folder to a GitHub repository.
2. In Cloudflare: Workers & Pages → Create → Pages → Connect to Git.
3. Framework preset: None.
4. Build command: leave blank.
5. Build output directory: `/` or leave default/root.
6. Deploy.

## Before AdSense

Replace `https://your-domain.com` in generated files with your real domain, update Contact/Privacy pages, add more original reviews, then apply.
